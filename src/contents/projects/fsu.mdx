## Description

The **FSU Web App** was developed as part of the Free Student Union IOE ERC Nepal Web App competition. This web app was built using **Next.js**, **React.js**, **Tailwind CSS**, and **Email.js**, ensuring a seamless and responsive user experience. It serves as a platform for students to interact, access resources, and communicate within the student union community.

## Features

- **Responsive Design**: The app is built with Tailwind CSS, making it fully responsive and mobile-friendly.
- **Interactive UI**: Uses React.js to create a dynamic and smooth user experience.
- **Email Integration**: Utilizes Email.js to enable direct communication via contact forms.
- **User-Friendly Interface**: Simple and intuitive UI, providing easy navigation for students.
