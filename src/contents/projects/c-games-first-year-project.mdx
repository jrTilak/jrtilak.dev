## Description

C Games is an interactive game collection developed for the C programming subject at IOE, Purwanchal Campus, Dharan. It features multiplayer and single-player games such as Tic Tac Toe, Truth or Dare, Rock Paper Scissors, and Coin Flip, along with account creation, login, and game history tracking.

## Features

- **User Account Creation & Login**: Users can create and log into their accounts, with unique usernames and game history tracking.
- **Multiplayer & Single-player Games**: Includes Tic Tac Toe, Truth or Dare, Rock Paper Scissors, and Coin Flip.
- **Game History**: Records game names, scores, and dates, allowing users to track their progress.
