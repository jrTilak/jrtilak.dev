Bumblebee 2.0 is a cutting-edge AI chatbot designed to offer intelligent, real-time responses to user queries. Built using the MERN stack, it incorporates Google's Gemini AI model to deliver highly context-aware interactions. Inspired by the Transformer character Bumblebee, the chatbot combines an engaging theme with advanced functionality for an enhanced user experience.

## Key Features

- **User-Friendly Interface**  
  Bumblebee 2.0 features a sleek, responsive design powered by React.js and TailwindCSS, ensuring smooth navigation across devices.

- **AI Integration**  
  Leveraging Google's Gemini AI model, the chatbot provides context-aware and meaningful responses, making it an effective assistant for diverse use cases.

- **Credit System**  
  Introduced a token-based system for chat interactions, allowing users to monitor and manage usage effectively.

- **Authorization System**  
  Implemented a secure login system with guest access functionality to provide seamless and personalized experiences.

- **Transformer-Inspired Design**  
  The chatbot adopts a Bumblebee-inspired theme, adding a creative and engaging touch to its visual and functional appeal.

## Development Highlights

- Developed the back-end using Express.js and MongoDB for scalable data management and seamless integration.
- Designed and optimized the front-end with React.js and TailwindCSS to ensure a modern and efficient UI/UX.
- Focused on secure user authentication and data privacy throughout the application.
