## Description

GemQuiz is a multiple-choice quiz app built using C++ and QT-Creator. The app allows users to take quizzes on various topics, track their scores, and enhance their knowledge in a fun and engaging way.

## Features

- **Multiple-Choice Quiz**: Users can select from different quiz topics, answer multiple-choice questions, and get instant feedback on their answers.
- **Score Tracking**: The app records the user's performance and displays the total score at the end of each quiz.
- **User-Friendly Interface**: Built with QT-Creator, the app features a simple and easy-to-navigate interface for an intuitive user experience.
